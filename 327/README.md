## B問題
- c++では
    ```c++
    std::pow(15, 15)
    ```
    で15の15乗を計算すると、誤差が発生する。
    これはpow関数ではdouble型が使用されており、数字が大きくなると誤差が生じるためである。

    long double型を扱える
    ```
    powl(15, 15)
    ```
    も存在するが今回は1つのテストケースでエラーが発生する。

    コンテストでc++で冪乗を扱うときはfor文で書くのが無難か。
